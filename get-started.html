get started

Parslet is a Parsing Expression Grammar (PEG) Parser. These are a different kind of grammars that recognize almost the same languages as your conventional LR parser, except that they are easier to work with, since they haven’t been conceived for generation, but for recognition of languages. You can read the founding paper of the field by Bryan Ford here.

 
$:.unshift File.dirname(__FILE__) # For use/testing when no gem is installed

# Require all of the Ruby files in the given directory.
#
# path - The String relative path from here to the directory.
#
# Returns nothing.
def require_all(path)
  glob = File.join(File.dirname(__FILE__), path, '*.rb')
  Dir[glob].each do |f|
    require f
  end
end
