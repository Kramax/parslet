Class Parslet::Source < Object

  • lib/parslet/source.rb

Wraps the input IO to parslet. The interface defined by this class is smaller than what IO offers, but enhances it with a column and line method for the current position.

    # File lib/parslet/source.rb, line 46
46:   def line_and_column(position=nil)
47:     pos = (position || self.pos)
48:     eol_idx = @line_ends.lbound(pos)
49:     
50:     if eol_idx
51:       # eol_idx points to the offset that ends the current line.
52:       # Let's try to find the offset that starts it: 
53:       offset = eol_idx>0 && @line_ends[eol_idx-1] || 0
54:       return [eol_idx+1, pos-offset+1]
55:     else
56:       # eol_idx is nil, that means that we're beyond the last line end that
57:       # we know about. Pretend for now that we're just on the last line.
58:       offset = @line_ends.last || 0
59:       return [@line_ends.size+1, pos-offset+1]
60:     end
61:   end
