Module Parslet::Source::RangeSearch

  • lib/parslet/source.rb

Mixin for arrays that implicitly give a number of ranges, where one range begins where the other one ends. 

  Example:

    [10, 20, 30]
    # would describe [0, 10], (10, 20], (20, 30]
