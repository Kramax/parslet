Parslet is at version _1.1.1_. 

*Rubygems*

{% highlight bash %} 
gem install parslet
{% endhighlight %}

*Bundler*

{% highlight bash %} 
gem 'parslet', '~> 1.1'
{% endhighlight %}

or if you want to track the edge: 

{% highlight bash %} 
gem 'parslet', :git => 'git://github.com/kschiess/parslet.git'
{% endhighlight %}
