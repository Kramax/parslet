Contribute

Find parslet to be really useful? Or just found a bug that is really ruining the day for you? Please take part by contributing:

Bugs

Log in to github and open a bug ticket here. Please be sure to include the version of parslet and Ruby; maybe you can even provide some code that exhibits the bug?

And of course if you provide a properly tested patch, you’ll be my hero and get a place in the space below for lifetime. Just send me a pull request!
